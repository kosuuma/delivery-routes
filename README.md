# 🗺️ Маршруты - Инструкция по размещению

## Способ 1: Netlify Drop (САМЫЙ БЫСТРЫЙ - 1 минута)

1. Откройте сайт: https://app.netlify.com/drop
2. Перетащите файл `index.html` в окно браузера
3. Дождитесь загрузки (10-20 секунд)
4. **Готово!** Скопируйте ссылку и отправьте кому нужно

**Плюсы:**
- Мгновенно (30 секунд)
- Не нужна регистрация
- Бесплатно навсегда
- Автоматический HTTPS

---

## Способ 2: GitHub Pages (Рекомендуется)

### Шаг 1: Регистрация
1. Зайдите на https://github.com
2. Нажмите "Sign up"
3. Создайте аккаунт

### Шаг 2: Создание репозитория
1. После входа нажмите зеленую кнопку "New" (слева)
2. Repository name: `delivery-routes` (или любое другое)
3. Выберите "Public"
4. Нажмите "Create repository"

### Шаг 3: Загрузка файла
1. Нажмите "uploading an existing file"
2. Перетащите файл `index.html` 
3. Нажмите "Commit changes"

### Шаг 4: Включение GitHub Pages
1. Перейдите в Settings (вверху справа)
2. Слева найдите "Pages"
3. В разделе "Source" выберите "main"
4. Нажмите "Save"

### Шаг 5: Получение ссылки
Через 1-2 минуты сайт будет доступен по адресу:
```
https://ваш-username.github.io/delivery-routes/
```

**Плюсы:**
- Бесплатно навсегда
- Можно обновлять файл
- Профессиональный хостинг
- Автоматический HTTPS

## Способ 3: Просто отправить файл

Если человек технически подкован, можно просто отправить файл `index.html`:

1. Отправьте файл через:
   - Email (прикрепите файл)
   - Telegram/WhatsApp (как документ)
   - Google Drive/Яндекс.Диск (поделитесь ссылкой)

2. Получатель просто откроет файл в браузере
   - Двойной клик по файлу
   - Или правой кнопкой → Открыть с помощью → Chrome/Firefox

**Минусы:**
- Нужно каждый раз отправлять новую версию
- Файл хранится локально
- Нет постоянной ссылки
